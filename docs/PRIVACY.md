# 🔐 Приватность

- отдельный сервер и база данных отсутствуют;
- кадровые и учебные данные находятся в текущей книге;
- исходные листы не переписываются;
- служебные листы могут быть скрыты, но скрытие не является контролем доступа;
- пользователь, имеющий доступ к книге, может открыть VBA, таблицы и исходные данные;
- для передачи книги используйте шифрование файла и корпоративные правила доступа;
- не публикуйте реальные книги в GitHub Issues.

Современная панель загружает Office.js с официального CDN Microsoft. Собственные данные проекта не отправляются в проектный backend.
