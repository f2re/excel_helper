## Практические рекомендации

- Для массовой очистки используйте апплеты: они записывают результат сразу для всего выделения.
- Для повторяемых расчётов используйте функции в ячейках: они пересчитываются вместе с книгой.
- Функции склонения ФИО и нечёткого сравнения используют эвристики. Неоднозначные результаты проверяйте вручную.
- Даты возвращаются как серийные значения Excel и форматируются стандартным форматированием ячейки.
