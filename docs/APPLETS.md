## Поведение апплетов

- Операции над выделением не заменяют исходные данные без явного действия; результат обычно создаётся рядом или на отдельном листе.
- Шаблоны идемпотентны: повторный запуск открывает существующий лист и не очищает введённые строки.
- Апплет `A23` автоматически создаёт отсутствующую служебную структуру книги.
- Для операций, зависящих от выделения, сначала выделите диапазон в Excel, затем запускайте апплет.
