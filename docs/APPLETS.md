# 🧩 Апплеты

В панели зарегистрированы **44 апплета**.

| Коды | Раздел | Апплеты |
|---|---|---|
| A01–A08 | Данные | Мастер ФИО, очистка контактов, дубли, нечёткое сопоставление, разделение столбцов, импорт, аудит, обезличивание |
| A09–A16 | Кадры | Кадровый реестр, штатное расписание, должности и ставки, договоры, отсутствия, табель, дни рождения, дашборд |
| A17–A22 | Рекрутинг | Воронка, дубли кандидатов, соответствие вакансии, собеседования, SLA, дашборд рекрутера |
| A23–A30 | Обучение | Составитель сводного расписания, конфликт-аудит, дубли занятий, нагрузка, посещаемость, оценки, аудитории, план курса |
| A31–A44 | Планирование | Недельный план, Канбан, Гант, статусы, карта загрузки, мастер формул, отчёты, шаблоны, экспорт, календарь, история, цепочки, аудит формул, справочники |

Ключевой апплет A23 запускается и в чистой книге: служебные страницы создаются автоматически, после чего пользователь подключает лист группы и подтверждает профиль парсера.
