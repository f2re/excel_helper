## Команды меню

- проверить или создать проект;
- составить сводное расписание;
- настроить парсер;
- нормализовать ФИО;
- подсветить дубли;
- создать недельный план.

## Функции в ячейках

Примеры:

```excel
=PROFI_FIO_NORMALIZE(A2)
=PROFI_FIO_SHORT(A2)
=PROFI_POSITION_COUNT(A2;$A$2:$A$500;$D$2:$D$500)
=PROFI_PHONE_NORMALIZE(B2)
=PROFI_TASK_STATUS(C2;D2)
```

## Первый запуск расписания

1. Откройте чистую книгу.
2. Надстройки → ПрофиПомощник → Составить сводное расписание.
3. Служебные таблицы будут созданы автоматически.
4. Откройте лист группы.
5. Выберите способ получения группы: выбранная ячейка, ручной ввод или имя листа.
6. Подтвердите найденные номера строк и столбцов.
7. Если структура определена неверно, измените значения в диалогах либо в `tblParserProfiles`.
8. Запустите составление повторно.

## Безопасность

Не оставляйте параметр «Доверять доступ к объектной модели VBA-проекта» включённым после сборки. Подпишите XLAM корпоративным сертификатом. Не размещайте файл в общедоступной сетевой папке с правом записи для всех пользователей.
