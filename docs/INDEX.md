## Основные понятия

| Термин | Значение |
|---|---|
| Апплет | Законченная операция над выделением или шаблон рабочего листа |
| Служебный лист | Скрытый лист `_PROFI_*`, используемый как структурированное хранилище проекта |
| Источник | Групповой лист расписания, зарегистрированный в `tblSources` |
| Профиль парсера | Координаты и правила разбора конкретного группового листа |
| Занятие | Нормализованная строка в `tblLessons` |
| Кандидат-преподаватель | Преподаватель из легенды, допустимый для дисциплины и вида занятия |
| Назначение | Связь занятия и преподавателя в `tblAssignments` |
| Поток | Одинаковое занятие нескольких групп в одном слоте, которое ведёт один преподаватель |
| Снимок | Версия назначений, сохранённая перед пересчётом |
