# 🧭 Руководство пользователя

## Поиск

В современной панели поиск охватывает 114 функций, 44 апплета и основные справочные команды.

## Функции

Выберите функцию, прочитайте описание и вставьте формулу в текущую ячейку. Каталог: [FUNCTIONS.md](FUNCTIONS.md).

## Апплеты

Апплеты выполняют массовые преобразования либо создают готовые рабочие листы. Каталог: [APPLETS.md](APPLETS.md).

## Расписание

1. откройте лист группы;
2. нажмите «Подключить активный лист»;
3. укажите группу вручную, по имени листа или через адрес ячейки;
4. проверьте `tblParserProfiles`;
5. нажмите «Разобрать и составить»;
6. устраните проблемы на листе `Контроль расписания`;
7. повторите расчёт.

Подробно: [SCHEDULE_WORKFLOW.md](SCHEDULE_WORKFLOW.md).

## Служебные листы

Обычно `_PROFI_*` листы скрыты. Команда «Показать служебные листы» открывает их для диагностики и ручной настройки. Удалённые таблицы восстанавливаются при следующей проверке структуры.
