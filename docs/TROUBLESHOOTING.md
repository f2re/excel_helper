# 🧯 Диагностика

## В `main` нет исходников

Проверьте наличие `package.json`, `src/`, `legacy-vba/`, `manifest.xml` и `manifest-office2016.xml`. Затем выполните `npm ci && npm run check`.

## Панель Office.js не открывается

- проверьте HTTPS и доверие сертификату;
- убедитесь, что URL манифеста доступен;
- для Excel 2016 используйте `manifest-office2016.xml`;
- если requirement set отсутствует, установите XLAM.

## `#NAME?` для `PROFI.*`

Custom Functions не поддерживаются данным клиентом либо не загружены метаданные. В старом Excel используйте VBA-функции `PROFI_*` из XLAM.

## PowerShell не собирает XLAM

- Excel должен быть установлен;
- включите доступ к объектной модели VBA-проекта;
- проверьте, что антивирус не блокирует COM Automation;
- соберите XLAM вручную по [INSTALLATION.md](INSTALLATION.md).

## Группа определилась неверно

Используйте ручной ввод или режим выбранной ячейки. В таблице `tblSources` можно изменить режим и адрес.

## Парсер не нашёл недели или легенду

Покажите служебные листы, откройте `tblParserProfiles`, исправьте строки, столбцы и смещения и запустите расчёт повторно.

## Все преподаватели не назначены

Заполните `tblTeachers`, варианты написания — `tblTeacherAliases`, затем проверьте столбцы легенды в профиле.
