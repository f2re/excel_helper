Что делает npm start

  1. Генерирует src/functions.json и справочники функций/апплетов.
  2. Собирает dist/ для https://localhost:3000.
  3. Создаёт локальный HTTPS-сертификат, если его ещё нет.
  4. Запускает HTTPS-сервер, слежение за исходниками и эту страницу.

Браузер не предоставляет объектную модель Excel. Для реальной работы с книгой загрузите dist/manifest.xml в Excel.
