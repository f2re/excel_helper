# 🧪 Примеры

## ФИО
```excel
=PROFI.ФИОНОРМА(A2)
=PROFI.ФИОКРАТКО(A2)
=PROFI.ФИОРАЗОБРАТЬ(A2)
=PROFI.ФИОСРАВНИТЬ(A2;B2)
```
## Кадры
```excel
=PROFI.ВОЗРАСТ(B2;СЕГОДНЯ())
=PROFI.СТАЖ(C2;СЕГОДНЯ();"text")
=PROFI.КОЛДОЛЖНОСТЕЙ(A2;$A$2:$A$500;$D$2:$D$500;ИСТИНА)
```
## Сводное расписание
Откройте панель «Расписание», выберите лист группы, укажите группу, проверьте распознанную структуру и запустите составление.
