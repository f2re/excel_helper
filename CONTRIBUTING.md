# 🤝 Участие в разработке

1. Создайте issue с описанием сценария.
2. Создайте ветку от `main`.
3. Добавьте код, тесты и документацию.
4. Выполните `npm run check`.
5. Откройте pull request с описанием пользовательского эффекта и результатов проверки.

Для нового шаблона расписания приложите обезличенную фикстуру или матрицу без персональных данных.
