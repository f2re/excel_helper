# 🧰 ПрофиПомощник для Excel

[![CI](https://github.com/f2re/excel_helper/actions/workflows/ci.yml/badge.svg)](https://github.com/f2re/excel_helper/actions/workflows/ci.yml)
[![Office Add-in](https://img.shields.io/badge/Excel-Office.js-217346)](https://learn.microsoft.com/office/dev/add-ins/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.9-3178C6)](https://www.typescriptlang.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**ПрофиПомощник** — надстройка Excel для кадровой работы, рекрутинга, преподавателей, сотрудников и учебного планирования. Она объединяет:

- **114 пользовательских функций** в ячейках с русскими именами;
- **44 апплета** для массовых операций и готовых рабочих листов;
- **мастер сводного расписания преподавателей**, перенесённый на сторону Excel из логики проекта [`planer-solving`](https://github.com/f2re/planer-solving);
- поиск по функциям и апплетам, встроенные подсказки, примеры и справку;
- самовосстанавливающуюся структуру служебных листов и таблиц.

> Надстройка не требует отдельного Python-сервиса, CLI или базы данных. Исходные групповые листы, промежуточные таблицы, назначения, конфликты и версии результата хранятся в текущей книге Excel.

## ✨ Ключевые возможности

| Раздел | Что входит |
|---|---|
| 👤 ФИО и кадры | Нормализация ФИО, инициалы, стаж, возраст, договоры, ставки, ИНН, СНИЛС, телефоны, кадровые шаблоны |
| 🎯 Рекрутинг | Дубли кандидатов, SLA, навыки, рейтинг, зарплатное и локационное соответствие, воронка, собеседования |
| 🎓 Обучение | Количество и часы занятий, академические часы, конфликты преподавателей, групп и аудиторий, посещаемость, оценки |
| ✅ Планирование | Статусы, дедлайны, прогресс, приоритеты, недельная загрузка, смены, план-факт, канбан и Гант |
| 🧹 Данные | Очистка текста, раскладка, транслитерация, даты и числа из текста, дубли, похожесть, списки, регулярные выражения |
| 📅 Сводное расписание | Импорт групповых листов, ручная корректировка парсера, легенда преподавателей, потоки, Round-Robin, аудит и итоговые листы |

Полные каталоги: [функции](docs/FUNCTIONS.md) · [апплеты](docs/APPLETS.md).

## 📅 Главный сценарий: сводное расписание из чистой книги

```mermaid
flowchart LR
    A[Чистая книга] --> B[Составить сводное расписание]
    B --> C[Автосоздание служебных листов и таблиц]
    C --> D[Импорт .xlsx/.xlsm или выбор существующего листа]
    D --> E[Группа вручную или из выбранной ячейки]
    E --> F[Автораспознавание сетки и легенды]
    F --> G{Распознано верно?}
    G -- Нет --> H[Исправить строки, столбцы, маркеры и смещения]
    H --> F
    G -- Да --> I[Сохранить источник]
    I --> J[Сопоставить преподавателей и алиасы]
    J --> K[Назначения: закрепления → сохранённые → потоки → Round-Robin]
    K --> L[Конфликты и контроль]
    L --> M[Сводный лист, неделя, преподаватель, нагрузка, аудитории]
```

Особенности мастера:

- при первом запуске автоматически создаёт все `_PROFI_*` листы и именованные таблицы;
- при последующем запуске восстанавливает удалённые служебные элементы, не очищая остальные данные;
- позволяет выбрать группу вручную или закрепить адрес ячейки, в которой она находится;
- показывает найденные координаты структуры до сохранения;
- разрешает вручную переопределить **каждую** строку, колонку, маркер и смещение парсера;
- хранит профиль разбора отдельно для каждого источника;
- скрывает исходный лист после сохранения, но не изменяет его содержимое;
- сохраняет снимок назначений перед каждым пересчётом.

Подробно: [Составитель сводного расписания](docs/SCHEDULE_COMPOSER.md).

## 🚀 Быстрый старт для разработчика

Требуются Node.js 20+, npm и OpenSSL.

```bash
git clone https://github.com/f2re/excel_helper.git
cd excel_helper
npm ci
npm run check
npm start
```

`npm start`:

1. собирает development-версию в `dist/`;
2. создаёт локальный HTTPS-сертификат в `.certs/`, если его нет;
3. запускает сервер на `https://localhost:3000`.

После доверия сертификату загрузите `dist/manifest.xml` в Excel. Пошаговые варианты для Excel в браузере, Windows, macOS и корпоративного развёртывания приведены в [руководстве по установке](docs/INSTALLATION.md).

## 🏗 Архитектура

```text
src/
├── actions/       # 44 апплета и шаблоны
├── commands/      # команды вкладки Excel
├── core/          # текст, даты, идентификаторы, общее состояние
├── functions/     # 114 функций и метаданные Excel
├── schedule/      # импорт, парсер, алиасы, назначения и рендереры
├── taskpane/      # боковая панель, поиск и мастера
└── workbook/      # служебная схема, таблицы и самовосстановление
```

Надстройка использует **shared runtime**: боковая панель, команды ленты и пользовательские функции загружаются в одном runtime. Это упрощает единое состояние и навигацию. Техническое описание: [Архитектура](docs/ARCHITECTURE.md).

## 💻 Совместимость

- основной целевой вариант: актуальный Microsoft 365 Excel для Windows, macOS и браузера;
- пользовательские функции Office.js не поддерживаются в Excel на iPad и в корпоративных бессрочных выпусках Office 2021 и старше;
- прямой импорт листов из внешней книги использует `insertWorksheetsFromBase64`; если API недоступен, мастер предлагает скопировать лист в текущую книгу и выбрать его вручную;
- в Excel для браузера импорт внешней книги может отклонить исходный лист со сводными таблицами, диаграммами, комментариями или срезами — в таком случае используется ручное копирование листа.

Актуальные ограничения Microsoft приведены со ссылками в [INSTALLATION.md](docs/INSTALLATION.md#-совместимость).

## 🔐 Данные и приватность

- отдельный сервер расписаний отсутствует;
- надстройка не отправляет кадровые или учебные данные в проектный backend;
- рабочие данные записываются в текущую книгу;
- исходные листы сохраняются и только скрываются;
- `Office.js` загружается с официального CDN Microsoft;
- для production статические файлы надстройки должны размещаться на доверенном HTTPS-хостинге.

Подробнее: [PRIVACY.md](docs/PRIVACY.md).
